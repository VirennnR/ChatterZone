# ChatterZone App
Realtime chat app with websockets using Node.js, Express and Socket.io with Vanilla JS on the frontend with a custom UI</br>
PS~ Frontend is not by me!

## Usage

npm install</br>
npm run dev

</br>
Go to localhost:4000
